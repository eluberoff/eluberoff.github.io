How to Play

Goal

Each die must move exactly its face value in steps and land on another die, without passing through any dice along the way. Aim for the lowest score.

Game Rules

Same values: Both dice disappear

Different values: Replace with the difference

Controls

Click: Select a die, then click adjacent squares to move

Drag: Select and drag through adjacent squares

Keyboard: Arrow keys to move, Tab to cycle dice, Escape to cancel
